Edit an announcement

Canvas allows instructors to edit announcements after they have been published.

  1. In Course Navigation, click the Announcements link.
  2. Click on the name of the announcement.
  3. Click the Edit button.
  4. Edit the necessary announcement fields. When finished, click the Publish button.
